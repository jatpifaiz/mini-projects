# Mini Projects

Kumpulan project frontend yang dibuat sebagai latihan membangun tampilan web menggunakan HTML, CSS, dan JavaScript. Semua produk dan brand yang ditampilkan merupakan **produk dummy (fiktif)** yang digunakan sebagai media latihan.

![projects](https://img.shields.io/badge/Projects-6-4ade80?style=flat-square&labelColor=1a1a2e) ![html](https://img.shields.io/badge/HTML-E34F26?style=flat-square&logo=html5&logoColor=white) ![css](https://img.shields.io/badge/CSS-1572B6?style=flat-square&logo=css3&logoColor=white) ![javascript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat-square&logo=javascript&logoColor=black)

## Description

Repository ini berisi beberapa project kecil yang masing-masing berfokus pada satu tema atau konsep tertentu. _Setiap project berdiri sendiri dan tidak ada shared dependency antar folder_.

Project-project ini tidak menggunakan backend atau database dan berjalan sepenuhnya di browser.

## Struktur Tiap Folder

```
nama-project/
├── assets/
│   └── images/
├── docs/               # screenshot preview untuk README
├── style/
│   └── style.css
├── js/                 # hanya untuk project yang menggunakan JavaScript
│   └── script.js
├── index.html
└── README.md
```

## How to Run

Clone repository:

```bash
git clone https://github.com/jatpifaiz/mini-projects.git
```

Masuk ke folder project yang ingin dijalankan, lalu buka file `index.html` di browser.
